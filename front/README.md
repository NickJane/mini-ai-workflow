# 新项目

基于 Vue 3 + TypeScript + Vite + Element Plus 的前端项目模板

## 功能特点

- ✅ 用户登录/注册
- ✅ Token 自动刷新
- ✅ 路由守卫
- ✅ 状态管理 (Pinia)

## 技术栈

- Vue 3
- TypeScript
- Vite
- Vue Router
- Pinia
- Element Plus
- Axios

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 后端配置

开发环境后端地址：`http://localhost:30050`

可在 `.env.development` 文件中修改。
